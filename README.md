<h1 align=center>NadekoConnector Client</h1>
<p align=center><b>A simple wrapper to request and get data from NadekoConnector endpoints.</b></p>

## Supported Endpoints

- getBotInfo
- getTables
- getFields
- execSql
- getCurrency
- setCurrency
- addCurrency
- subtractCurrency
- createTransaction
- getTransactions
- getGuildRank
- getGuildXp
- setGuildXp
- addGuildXp
- subtractGuildXp
- awardGuildXp
- getGuildXpLeaderboard
- getGuildXpRoleRewards
- getGuildXpCurrencyRewards
- getGlobalRank
- getGlobalXp
- getGlobalXpLeaderboard
- getClubLeaderboard
- getClubInfo
- getClubInfoByUser
- getClubMembers
